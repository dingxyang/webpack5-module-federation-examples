# dynamic-tabs
初始化菜单数据，点击菜单再去动态加载子系统

- `main` contains the "Home Page" (/) route.
- `child` contains the "About Page" (/about)  "Hello Page" (/hello) route.
- `child2` contains the "Test Page" (/test) route.

```
cd main
yarn run build
node serve.js
```
```
cd child1
yarn start
```
```
cd child2
yarn start
```
